SEP 11 Freedom Project

This was a year long project that we made using firebase and javascript. Me and Jimmy made a workout tracker and used firebase tools to save data from the workout tracker into the database

Firebase is is a set of backend cloud computing services and application development platforms provided by Google. It hosts databases, services, authentication, and integration for a variety of applications.

You would have to make an account and login using that account to access the workout tracker. The workout tracker is dynamic so it works for everyone for different users and wont show the same workouts on everyones page.

I plan on next moving this workout tracker to the phone and make this as an app for next years freedom project in APCSA. The main challenge of this project was importing firebase and learning how firebase imports data to the database and learning how to bring it out. Now finally learning how to do it, we can utilize the database and bring it to the app.
